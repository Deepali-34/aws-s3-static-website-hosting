## Project Overview

This project demonstrates the deployment of a **multi-page static website** using **Amazon S3 Static Website Hosting**. The implementation includes configuring an Amazon S3 bucket, enabling static website hosting, managing public access through Bucket Policies, and securing administrative permissions with AWS Identity and Access Management (IAM).

The project also follows professional software engineering practices by maintaining a well-structured repository, comprehensive technical documentation, and a clean deployment workflow.

# Deployment Workflow

1. Create an Amazon S3 Bucket.
2. Upload website source files.
3. Enable Static Website Hosting.
4. Configure Bucket Policy.
5. Configure Public Access.
6. Obtain the Website Endpoint.
7. Verify successful deployment.
