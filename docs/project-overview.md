# Project Overview

## Introduction

The AWS S3 Static Website Hosting project demonstrates how to deploy a static website using Amazon Web Services (AWS). The project focuses on hosting a website securely and efficiently by utilizing Amazon S3 as a scalable object storage service with Static Website Hosting enabled.

This implementation showcases the fundamental concepts of cloud computing, including cloud storage, access management, bucket policies, and website deployment without requiring a traditional web server.

---

## Project Objectives

- Deploy a static website using Amazon S3.
- Configure Static Website Hosting.
- Implement secure bucket access using Bucket Policies.
- Manage permissions through AWS Identity and Access Management (IAM).
- Make the website publicly accessible through the S3 Website Endpoint.
- Gain practical experience with AWS cloud services.

---

## Project Scope

The project covers the complete deployment lifecycle of a static website, including:

- Creating an Amazon S3 bucket
- Uploading website source files
- Configuring Static Website Hosting
- Applying Bucket Policies
- Managing Public Access settings
- Deploying the website
- Verifying public accessibility
